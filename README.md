# Apex Hikers 🏔️
> **Boutique High-Altitude Expedition Platform**

Apex Hikers is a premium, highly immersive digital platform designed for modern explorers seeking prestigious trekking expeditions across the Himalayas, Dolomites, Swiss Alps, and beyond. Rebranded and built from the ground up to showcase top-tier alpine journeys, this platform combines luxurious editorial design with interactive gameplay and intelligent planning tools.

## 🛠️ Tech Stack
- **Framework**: [Next.js 16](https://nextjs.org/) (App Router, dynamic preloading)
- **Core Library**: [React 19](https://react.dev/) (Concurrent rendering, Suspense APIs)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (Modern CSS engine, utility-first)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) (Smooth cross-fades, layout transitions, hover states)
- **Smooth Scrolling**: [Lenis](https://lenis.darkroom.engineering/) (Premium web physics-based kinetic scrolling)
- **Database / Backend**: [Supabase](https://supabase.com/) (Scale-ready postgres backend integration)
- **Language**: [TypeScript](https://www.typescriptlang.org/) (Type-safe compilation and structures)
- **Icons**: [Lucide React](https://lucide.dev/) (Crisp, modern SVG icons)

## ✨ Key Features & Achievements

### 🌟 1. Automatic Hero Fade Transitions
The Hero section utilizes an advanced cinematic multi-slide background controller. It automatically cross-fades every **10,000ms** using Framer Motion's `AnimatePresence`. 
- Incorporates a continuous, slow **Ken Burns scaling effect** (zooming from `1.1x` to `1.25x` over 20 seconds) on high-resolution alpine backdrops.
- Leverages magnetic buttons, floating typography, and backdrop glassmorphism to immediately capture high-intent users with a premium, alive look.

### 🎮 2. Interactive Season-Switching Gameplay
We designed an engaging interactive "Atmosphere Gameplay" in the Seasonal Spotlight section. Instead of simple standard buttons, visitors are presented with a text-input interface:
- Typing **"winter"** (or **"summer"**) in the seasonal search input dynamically triggers an elegant, sweeping transition.
- The entire atmosphere—including large background images, high-contrast descriptions, labels, and adventure CTA buttons—fades and switches seamlessly to represent either a sun-drenched alpine meadow or a snow-capped winter kingdom.

### 📰 3. Summit Journal & Explorer Blogs
The **Summit Journal** acts as an authority & educational hub, housing real-life high-altitude chronicles, expedition guides, and journals:
- Features immersive, editorial articles (e.g., *“Kedarkantha Trek: The Complete 2026 Guide”*, *“When to Visit Dayara Bugyal?”*) complete with large-drop capital letters, reading times, publication dates, and author profiles.
- Designed with professional grid styling and card hover transformations (grayscale-to-color transition and scale transitions) to encourage deep content exploration.

### 🧭 4. Multi-Dimensional Expedition Navigator
Finding the perfect adventure is effortless with the custom-built **Expedition Navigator** matrix:
- We have organized all trekking routes into beautiful, structured categories to help users select treks exactly according to their requirements:
  - **By Month**: Grouped calendars (e.g. *Jan - Mar*, *Apr - Jun*) for structured seasonal booking.
  - **By Season**: Dynamic listings filtering *Winter*, *Spring*, *Summer*, *Monsoon*, *Autumn*, *Alpine*, and *Pass Crossings*.
  - **By Difficulty**: Easy-to-understand progression scale (*Easy*, *Easy to Moderate*, *Moderate*, *Moderate to Difficult*, *Difficult*, *Challenging*).
  - **By Nature & Duration**: Instantly discover *Lake Treks*, *Summit Treks*, *Snow Treks*, or match trips by duration ranges (*2-3 Days*, *4-5 Days*, *6 Days*, *7+ Days*).

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/krrishvimal/Apexhikers.git
cd Apexhikers
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

### 4. Build for production
```bash
npm run build
```
